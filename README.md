Soft Tissue Sarcoma Clinical Analytics Dashboard
Project Overview

This project presents an interactive Power BI dashboard designed to analyze clinical data from patients diagnosed with Soft Tissue Sarcoma (STS). The dashboard supports clinical decision-making by visualizing patient demographics, treatment patterns, metastasis, and survival outcomes.

Objectives
Analyze patient demographics
Explore treatment patterns
Examine metastatic spread
Evaluate survival outcomes
Support evidence-based clinical decisions
